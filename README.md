# Taller TypeScript

Proyecto con 10 ejercicios prácticos en TypeScript.

## Estructura

```
src/
├── ejercicio1/
├── ejercicio2/
├── ejercicio3/
...
└── ejercicio10/
```

## Ejecución

Para ejecutar un ejercicio:

```
npx ts-node src/ejercicioX/ejercicioX.ts
```

Ejemplo:

```
npx ts-node src/ejercicio1/ejercicio1.ts
```
